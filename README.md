🍷 Wine Purchasing Behaviour Analysis (PCA + Lasso)
📌 Overview

Analyzed customer purchasing behavior using PCA and Lasso regression to identify key drivers of wine purchases and evaluate the relative importance of behavioral vs demographic factors.

🧠 Methods
Dimensionality Reduction: Principal Component Analysis (PCA)
Predictive Modeling: Lasso Regression
Dataset: 2,240 customer records, 42 features
📊 Results
R²: ~0.52–0.61
RMSE: 0.461
Reduced 22 variables → 11 principal components (95% variance retained)
🔍 Key Insight

Customer behavior (engagement, multi-channel activity) is a stronger predictor of wine purchases than demographic attributes.

🛠 Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
